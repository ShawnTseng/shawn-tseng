# Project Overview

This project is a portfolio showcasing my professional experiences and skills as a software engineer. It includes detailed descriptions of my work history, projects, and contributions across various companies.

## Features
- **Interactive Timeline**: Displays work experiences with detailed information about roles, projects, and contributions.
- **Video Demonstrations**: Includes video links to showcase project highlights.
- **Technology Stack**: Highlights frameworks, tools, and languages used in each role.
